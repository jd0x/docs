Deleting a cluster
==================

To delete a cluster, use the ``hasura`` CLI.

.. code-block:: bash

  # to delete cluster named 'alarming52'
  $ hasura cluster delete absolve54

  This action will permanently delete cluster:
    absolve54
  You CANNOT undo this action. Proceed? (y/n) yes
